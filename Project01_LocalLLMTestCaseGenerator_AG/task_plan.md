# Task Plan

## Goals
- Build an LLM-powered Test Case Generator supporting multiple LLM providers (Ollama, LM Studio, OpenAI, Grok, Claude).
- Accept Jira requirements/text inputs via chat or copy-paste.
- Generate both API and Web Application test cases (functional and non-functional) formatted specifically for Jira.

## Approved Blueprint
1. **Tech Stack**: Backend: Node.js with TypeScript. Frontend: React.js.
2. **LLM Integrations**: 
   - Local: Ollama API, LM Studio
   - Cloud: OpenAI API, Grok API, Claude API
3. **Inputs**: Chat interface or Text area for users to paste Jira requirements.
4. **Outputs**: Structured test case generation for API & Web (Functional/Non-functional) in Jira format.
